DATABASE DESIGN:
================

TABLE: users
  Fields: id, first_name, middle_name, last_name, email, 
          student_id, department, password, has_voted, registration_date
  Unique Constraints: email, student_id
  Indexes: idx_student_id, idx_email

TABLE: votes
  Fields: vote_id, student_id, president, 
          vice_president, secretary, timestamp
  Foreign Key: student_id → users.student_id
  Indexes: idx_vote_student, idx_vote_timestamp

SECURITY FEATURES IMPLEMENTED:
================================

✓ Password Security
  - Uses PHP password_hash() with PASSWORD_DEFAULT
  - Verified with password_verify()
  - Minimum 6 characters requirement
  - Securely stored in database

✓ SQL Injection Prevention
  - All queries use prepared statements
  - Parameterized queries with bind_param()
  - No direct string concatenation in SQL

✓ Session Management
  - PHP sessions for user authentication
  - Session variables: user_id, student_id, first_name, has_voted
  - Session destruction on logout
  - Automatic redirect if not logged in

✓ One-Vote Prevention
  - has_voted database flag (0 or 1)
  - Double-check before vote submission
  - Redirect to already_voted.php if voted

✓ Input Validation & Sanitization
  - Email format validation
  - Required field checks
  - HTML escaping with htmlspecialchars()
  - Trim whitespace

✓ XSS Protection
  - All user output escaped
  - No direct HTML injection possible

MANAGEMENT & MAINTENANCE:
=========================

📊 Monitor Database:
   SELECT COUNT(*) as total_users FROM users;
   SELECT COUNT(*) as total_votes FROM votes;
   SELECT COUNT(*) as voted_users FROM users WHERE has_voted = 1;

🔧 User Management:
   - Reset user voting status: UPDATE users SET has_voted = 0 WHERE student_id = 'XXX';
   - Delete user: DELETE FROM users WHERE student_id = 'XXX';
   - View specific user: SELECT * FROM users WHERE student_id = 'XXX';

📈 Vote Counting:
   SELECT president, COUNT(*) as votes FROM votes GROUP BY president;
   SELECT vice_president, COUNT(*) as votes FROM votes GROUP BY vice_president;
   SELECT secretary, COUNT(*) as votes FROM votes GROUP BY secretary;

💾 Backup:
   mysqldump -u root voting_system > backup.sql
