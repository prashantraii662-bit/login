DATABASE SCHEMA:
═══════════════════════════════════════════════════════════════════════════════

TABLE: users
├─ id (INT, PRIMARY KEY, AUTO_INCREMENT)
├─ first_name (VARCHAR 50, NOT NULL)
├─ middle_name (VARCHAR 50)
├─ last_name (VARCHAR 50, NOT NULL)
├─ email (VARCHAR 100, UNIQUE, NOT NULL)
├─ student_id (VARCHAR 20, UNIQUE, NOT NULL)
├─ department (VARCHAR 100, NOT NULL)
├─ password (VARCHAR 255, NOT NULL - hashed)
├─ has_voted (INT, DEFAULT 0)
└─ created_at (TIMESTAMP, DEFAULT CURRENT)

TABLE: votes
├─ vote_id (INT, PRIMARY KEY, AUTO_INCREMENT)
├─ student_id (VARCHAR 20, FOREIGN KEY → users)
├─ president (VARCHAR 100, NOT NULL)
├─ vice_president (VARCHAR 100, NOT NULL)
├─ secretary (VARCHAR 100, NOT NULL)
└─ timestamp (TIMESTAMP, DEFAULT CURRENT)

TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════════

Problem: "Database Connection Failed"
Solution: 
   • Check MySQL is running (XAMPP Control Panel)
   • Verify credentials in db.php match your setup
   • Ensure database "voting_system" exists

Problem: "Table doesn't exist"
Solution:
   • Execute database.sql in phpMyAdmin
   • Check that tables appear in phpMyAdmin
   • Use phpmyadmin to verify table structure

Problem: "User registered but can't login"
Solution:
   • Verify user is in database (phpMyAdmin → users table)
   • Check password hash is stored correctly
   • Try restarting MySQL

Problem: "Vote won't submit"
Solution:
   • Ensure all three positions are selected
   • Check browser console for JavaScript errors
   • Verify has_voted flag in database

Problem: "Redirect loops"
Solution:
   • Clear browser cookies
   • Restart PHP session
   • Check session configuration
