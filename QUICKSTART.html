🚀 College Online Voting System - Quick Start Guide

📋 Step-by-Step Setup

Step 1: Extract Files

Place the project folder in your XAMPP htdocs directory:

C:\xampp\htdocs\Voting\

Step 2: Start XAMPP Services

Open XAMPP Control Panel and start:

  • ✓ Apache (Required for PHP)
  • ✓ MySQL (Required for Database)

Step 3: Create Database

Option A: Using phpMyAdmin (Recommended)

  1. Go to: http://localhost/phpmyadmin
  2. Click on "SQL" tab
  3. Copy contents of database_setup.sql
  4. Paste and execute

Option B: Using Command Line

mysql -u root < database_setup.sql

Step 4: Verify Configuration

Open config.php and verify:

Setting Default Value Change If Needed
Servername localhost Usually correct
Username root If different
Password (empty) If MySQL has password
Database voting_system If different name

Step 5: Launch the Application

Open your browser and go to:

http://localhost/Voting/index.html

✓ You should see the landing page!

🧪 Testing the System

Test Registration

  1. Go to: http://localhost/Voting/register.php
  2. Fill in all fields with test data
  3. Use Student ID like: TEST001
  4. Create a password (min 6 characters)
  5. Click "Register"

Test Login & Voting

  1. Go to: http://localhost/Voting/login.php
  2. Enter your Student ID and password
  3. Click "Login"
  4. Select candidates and submit vote
  5. Try logging in again - should see "Already Voted" message

🔍 Verify Database

To check if registration and votes were saved:

View Users:

SELECT * FROM users;

View Votes:

SELECT * FROM votes;

⚙️ Configuration Notes

⚠️ Common Issues & Fixes

Issue: "Connection failed"

  • ✓ Check if MySQL is running
  • ✓ Verify database credentials in config.php
  • ✓ Ensure database_setup.sql was executed

Issue: "Student ID already exists"

  • ✓ This is correct behavior - Student IDs must be unique
  • ✓ Try registering with a different Student ID

Issue: Blank page when visiting PHP files

  • ✓ Ensure Apache is running in XAMPP
  • ✓ Check browser console for errors (F12)
  • ✓ Verify file paths are correct

Issue: Can't access http://localhost

  • ✓ Start Apache from XAMPP Control Panel
  • ✓ Wait 10 seconds for Apache to fully start
  • ✓ Check if port 80 is available

🔒 Security Reminders

  • ✓ Passwords are hashed with PHP's password_hash()
  • ✓ All SQL queries use prepared statements (prevent SQL injection)
  • ✓ Sessions are used for secure authentication
  • ✓ One-vote prevention with database flag
  • ✓ Input validation and HTML escaping on all outputs
